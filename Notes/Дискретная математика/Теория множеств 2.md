#### –ê–∫—Å–∏–æ–º—ã ZFC (Zermelo-Fraenkel with Choice)

ZFC ‚Äî —ç—Ç–æ –∞–∫—Å–∏–æ–º–∞—Ç–∏—á–µ—Å–∫–∞—è —Å–∏—Å—Ç–µ–º–∞, —è–≤–ª—è—é—â–∞—è—Å—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–º —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–æ–º –¥–ª—è —Å–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–π —Ç–µ–æ—Ä–∏–∏ –º–Ω–æ–∂–µ—Å—Ç–≤. –û—Å–Ω–æ–≤–Ω—ã–µ –∞–∫—Å–∏–æ–º—ã:

1.  **–ê–∫—Å–∏–æ–º–∞ –æ–±—ä—ë–º–Ω–æ—Å—Ç–∏ (Extensionality):** –î–≤–∞ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ä–∞–≤–Ω—ã —Ç–æ–≥–¥–∞ –∏ —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ –æ–Ω–∏ –∏–º–µ—é—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã.
    `‚àÄA, B. (‚àÄx. x ‚àà A ‚áî x ‚àà B) ‚áí A = B`

2.  **–ê–∫—Å–∏–æ–º–∞ –ø—É—Å—Ç–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ (Empty Set):** –°—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ, –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤.
    `‚àÉ‚àÖ. ‚àÄx. x ‚àâ ‚àÖ`

3.  **–ê–∫—Å–∏–æ–º–∞ –ø–∞—Ä—ã (Pairing):** –î–ª—è –ª—é–±—ã—Ö –¥–≤—É—Ö –º–Ω–æ–∂–µ—Å—Ç–≤ `a` –∏ `b` —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ `{a, b}`, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ –∏—Ö –≤ –∫–∞—á–µ—Å—Ç–≤–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤.
    `‚àÄa, b. ‚àÉC. ‚àÄx. x ‚àà C ‚áî (x = a ‚à® x = b)`

4.  **–ê–∫—Å–∏–æ–º–∞ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è (Union):** –î–ª—è –ª—é–±–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ `F` (—Å–æ–¥–µ—Ä–∂–∞—â–µ–≥–æ –¥—Ä—É–≥–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤–∞) —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ `U`, —è–≤–ª—è—é—â–µ–µ—Å—è –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ–º –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ `F`.
    `‚àÄF. ‚àÉU. ‚àÄx. x ‚àà U ‚áî ‚àÉA ‚àà F. x ‚àà A`

5.  **–ê–∫—Å–∏–æ–º–∞ —Å—Ç–µ–ø–µ–Ω–∏ (Power Set):** –î–ª—è –ª—é–±–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ `A` —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ `P(A)`, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ –≤—Å–µ –µ–≥–æ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞.
    `‚àÄA. ‚àÉP(A). ‚àÄX. X ‚àà P(A) ‚áî X ‚äÜ A`

6.  **–ê–∫—Å–∏–æ–º–∞ –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ—Å—Ç–∏ (Infinity):** –°—É—â–µ—Å—Ç–≤—É–µ—Ç –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ. (–ù–∞–ø—Ä–∏–º–µ—Ä, –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ –Ω–∞—Ç—É—Ä–∞–ª—å–Ω—ã–µ —á–∏—Å–ª–∞, –ø–æ—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –∏–Ω–¥—É–∫—Ç–∏–≤–Ω–æ: `‚àÖ, {‚àÖ}, {‚àÖ, {‚àÖ}}, ...`).
    `‚àÉS. ‚àÖ ‚àà S ‚àß (‚àÄx ‚àà S. x ‚à™ {x} ‚àà S)`

7.  **–ê–∫—Å–∏–æ–º–∞ –≤—ã–¥–µ–ª–µ–Ω–∏—è (–°—Ö–µ–º–∞ –∞–∫—Å–∏–æ–º –ø–æ–¥—Å—Ç–∞–Ω–æ–≤–∫–∏) (Separation/Replacement):** –î–ª—è –ª—é–±–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ `A` –∏ –ª—é–±–æ–≥–æ —Å–≤–æ–π—Å—Ç–≤–∞ `P` –º–æ–∂–Ω–æ –æ–±—Ä–∞–∑–æ–≤–∞—Ç—å –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ `{x ‚àà A | P(x)}`. *–≠—Ç–∞ –∞–∫—Å–∏–æ–º–∞ –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–∞–µ—Ç –ø–∞—Ä–∞–¥–æ–∫—Å –†–∞—Å—Å–µ–ª–∞, –æ–≥—Ä–∞–Ω–∏—á–∏–≤–∞—è –æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤ —É–∂–µ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–º–∏.*

8.  **–ê–∫—Å–∏–æ–º–∞ —Ñ—É–Ω–¥–∏—Ä–æ–≤–∞–Ω–∏—è (Regularity):** –ö–∞–∂–¥–æ–µ –Ω–µ–ø—É—Å—Ç–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ `A` —Å–æ–¥–µ—Ä–∂–∏—Ç —ç–ª–µ–º–µ–Ω—Ç `x`, –∫–æ—Ç–æ—Ä—ã–π –Ω–µ –∏–º–µ–µ—Ç –æ–±—â–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å `A`. –≠—Ç–∞ –∞–∫—Å–∏–æ–º–∞ –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–∞–µ—Ç —Ç–∞–∫–∏–µ –ø–∞—Ä–∞–¥–æ–∫—Å—ã, –∫–∞–∫ `x ‚àà x`.
    `‚àÄA. A ‚â† ‚àÖ ‚áí ‚àÉx ‚àà A. A ‚à© x = ‚àÖ`

9.  **–ê–∫—Å–∏–æ–º–∞ –≤—ã–±–æ—Ä–∞ (Choice):** –î–ª—è –ª—é–±–æ–≥–æ —Å–µ–º–µ–π—Å—Ç–≤–∞ `F` –Ω–µ–ø—É—Å—Ç—ã—Ö –Ω–µ–ø–µ—Ä–µ—Å–µ–∫–∞—é—â–∏—Ö—Å—è –º–Ω–æ–∂–µ—Å—Ç–≤ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç —Ñ—É–Ω–∫—Ü–∏—è –≤—ã–±–æ—Ä–∞ `f`, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–±–∏—Ä–∞–µ—Ç –ø–æ –æ–¥–Ω–æ–º—É —ç–ª–µ–º–µ–Ω—Ç—É –∏–∑ –∫–∞–∂–¥–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –≤ `F`.
    `‚àÄF. (‚àÖ ‚àâ F) ‚áí ‚àÉf. ‚àÄA ‚àà F. f(A) ‚àà A`

#### –ü–æ–ø–∞—Ä–Ω–æ –Ω–µ–ø–µ—Ä–µ—Å–µ–∫–∞—é—â–∏–µ—Å—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞ & –†–∞–∑–±–∏–µ–Ω–∏–µ (Partition)

*   **–ü–æ–ø–∞—Ä–Ω–æ –Ω–µ–ø–µ—Ä–µ—Å–µ–∫–∞—é—â–∏–µ—Å—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞:** –ö–æ–ª–ª–µ–∫—Ü–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤ `{A‚ÇÅ, A‚ÇÇ, ..., A‚Çô}` —è–≤–ª—è–µ—Ç—Å—è –ø–æ–ø–∞—Ä–Ω–æ –Ω–µ–ø–µ—Ä–µ—Å–µ–∫–∞—é—â–µ–π—Å—è, –µ—Å–ª–∏ –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ –ª—é–±—ã—Ö –¥–≤—É—Ö —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –º–Ω–æ–∂–µ—Å—Ç–≤ –ø—É—Å—Ç–æ: `A·µ¢ ‚à© A‚±º = ‚àÖ` –¥–ª—è –≤—Å–µ—Ö `i ‚â† j`.

*   **–†–∞–∑–±–∏–µ–Ω–∏–µ (Partition):** –†–∞–∑–±–∏–µ–Ω–∏–µ–º –º–Ω–æ–∂–µ—Å—Ç–≤–∞ `M` –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –∫–æ–ª–ª–µ–∫—Ü–∏—è –ø–æ–ø–∞—Ä–Ω–æ –Ω–µ–ø–µ—Ä–µ—Å–µ–∫–∞—é—â–∏—Ö—Å—è *–Ω–µ–ø—É—Å—Ç—ã—Ö* –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤ `M`, –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ –∫–æ—Ç–æ—Ä—ã—Ö –¥–∞—ë—Ç –≤—Å—ë `M`.
    –§–æ—Ä–º–∞–ª—å–Ω–æ, `ùí´ = {A‚ÇÅ, A‚ÇÇ, ..., A‚Çô}` —è–≤–ª—è–µ—Ç—Å—è —Ä–∞–∑–±–∏–µ–Ω–∏–µ–º `M`, –µ—Å–ª–∏:
    1.  `A·µ¢ ‚â† ‚àÖ` –¥–ª—è –≤—Å–µ—Ö `i`
    2.  `A·µ¢ ‚à© A‚±º = ‚àÖ` –¥–ª—è –≤—Å–µ—Ö `i ‚â† j`
    3.  `A‚ÇÅ ‚à™ A‚ÇÇ ‚à™ ... ‚à™ A‚Çô = M`

    *–ü—Ä–∏–º–µ—Ä:* `ùí´ = {{1,3}, {2,6}, {4,5}}` —è–≤–ª—è–µ—Ç—Å—è —Ä–∞–∑–±–∏–µ–Ω–∏–µ–º `M = {1,2,3,4,5,6}`.

#### –î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ –º–æ—â–Ω–æ—Å—Ç–∏ —Å—Ç–µ–ø–µ–Ω–Ω–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ (–ë—É–ª–µ–∞–Ω–∞) –ø–æ –∏–Ω–¥—É–∫—Ü–∏–∏

*   **–£—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–µ:** –î–ª—è –ª—é–±–æ–≥–æ –∫–æ–Ω–µ—á–Ω–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ `A` –º–æ—â–Ω–æ—Å—Ç—å –µ–≥–æ —Å—Ç–µ–ø–µ–Ω–Ω–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ (–±—É–ª–µ–∞–Ω–∞) `P(A)` —Ä–∞–≤–Ω–∞ `2^{|A|}`.
*   **–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ (–ø–æ –∏–Ω–¥—É–∫—Ü–∏–∏ –ø–æ `n = |A|`):**
    *   **–ë–∞–∑–∞ –∏–Ω–¥—É–∫—Ü–∏–∏ (`n=0`):** `A = ‚àÖ`. `P(‚àÖ) = {‚àÖ}`. `|P(‚àÖ)| = 1 = 2‚Å∞`.
    *   **–ò–Ω–¥—É–∫—Ü–∏–æ–Ω–Ω–æ–µ –ø—Ä–µ–¥–ø–æ–ª–æ–∂–µ–Ω–∏–µ:** –ü—É—Å—Ç—å –¥–ª—è –ª—é–±–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ä–∞–∑–º–µ—Ä–∞ `k` —É—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–µ –≤–µ—Ä–Ω–æ: `|P(A)| = 2·µè`.
    *   **–ò–Ω–¥—É–∫—Ü–∏–æ–Ω–Ω—ã–π —à–∞–≥ (`n = k+1`):** –í–æ–∑—å–º—ë–º –º–Ω–æ–∂–µ—Å—Ç–≤–æ `A` —Ä–∞–∑–º–µ—Ä–∞ `k+1`. –í—ã–±–µ—Ä–µ–º –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç `a ‚àà A` –∏ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –º–Ω–æ–∂–µ—Å—Ç–≤–æ `A' = A \ {a}` —Ä–∞–∑–º–µ—Ä–∞ `k`.
        –í—Å–µ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞ `A` –º–æ–∂–Ω–æ —Ä–∞–∑–±–∏—Ç—å –Ω–∞ –¥–≤–∞ *–Ω–µ–ø–µ—Ä–µ—Å–µ–∫–∞—é—â–∏—Ö—Å—è* –∫–ª–∞—Å—Å–∞:
        1.  –ü–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞, –∫–æ—Ç–æ—Ä—ã–µ *–Ω–µ* —Å–æ–¥–µ—Ä–∂–∞—Ç `a`. –≠—Ç–æ –≤ —Ç–æ—á–Ω–æ—Å—Ç–∏ `P(A')`. –ü–æ –ø—Ä–µ–¥–ø–æ–ª–æ–∂–µ–Ω–∏—é –∏–Ω–¥—É–∫—Ü–∏–∏, –∏—Ö `2·µè`.
        2.  –ü–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞, –∫–æ—Ç–æ—Ä—ã–µ —Å–æ–¥–µ—Ä–∂–∞—Ç `a`. –ö–∞–∂–¥–æ–µ —Ç–∞–∫–æ–µ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏–º–µ–µ—Ç –≤–∏–¥ `S ‚à™ {a}`, –≥–¥–µ `S ‚äÜ A'`. –ú–µ–∂–¥—É —ç—Ç–∏–º –∫–ª–∞—Å—Å–æ–º –∏ `P(A')` —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –±–∏–µ–∫—Ü–∏—è, –∑–Ω–∞—á–∏—Ç, –∏—Ö —Ç–æ–∂–µ `2·µè`.
        –û–±—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤: `|P(A)| = 2·µè + 2·µè = 2 * 2·µè = 2^{k+1} = 2^{|A|}`. ‚àé

#### –î–∏–∞–≥—Ä–∞–º–º—ã –í–µ–Ω–Ω–∞ (Venn Diagrams)

*   **–î–∏–∞–≥—Ä–∞–º–º–∞ –í–µ–Ω–Ω–∞** ‚Äî —ç—Ç–æ –≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∞—è –∏–ª–ª—é—Å—Ç—Ä–∞—Ü–∏—è, –∏—Å–ø–æ–ª—å–∑—É–µ–º–∞—è –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è –≤—Å–µ—Ö –≤–æ–∑–º–æ–∂–Ω—ã—Ö –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö –æ—Ç–Ω–æ—à–µ–Ω–∏–π –º–µ–∂–¥—É –∫–æ–Ω–µ—á–Ω–æ–π –∫–æ–ª–ª–µ–∫—Ü–∏–µ–π –º–Ω–æ–∂–µ—Å—Ç–≤.
*   –ö–∞–∂–¥–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç—Å—è –≤ –≤–∏–¥–µ –∫—Ä—É–≥–∞. –û–±–ª–∞—Å—Ç–∏ –≤–Ω—É—Ç—Ä–∏ –∫—Ä—É–≥–æ–≤ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –º–Ω–æ–∂–µ—Å—Ç–≤.
*   –ü–µ—Ä–µ–∫—Ä—ã–≤–∞—é—â–∏–µ—Å—è –æ–±–ª–∞—Å—Ç–∏ –∫—Ä—É–≥–æ–≤ –ø–æ–∫–∞–∑—ã–≤–∞—é—Ç –æ–±—â–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã –º–µ–∂–¥—É –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏ (–ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ).
*   –û–±–ª–∞—Å—Ç–∏, –Ω–µ –≤—Ö–æ–¥—è—â–∏–µ –Ω–∏ –≤ –æ–¥–∏–Ω –∫—Ä—É–≥, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —ç–ª–µ–º–µ–Ω—Ç—ã, –Ω–µ –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–∞—â–∏–µ –Ω–∏ –æ–¥–Ω–æ–º—É –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤ (–¥–æ–ø–æ–ª–Ω–µ–Ω–∏–µ –¥–æ —É–Ω–∏–≤–µ—Ä—Å—É–º–∞).
*   *–û—Ç–ª–∏—á–∏–µ –æ—Ç –¥–∏–∞–≥—Ä–∞–º–º –≠–π–ª–µ—Ä–∞:* –î–∏–∞–≥—Ä–∞–º–º—ã –í–µ–Ω–Ω–∞ –ø–æ–∫–∞–∑—ã–≤–∞—é—Ç *–≤—Å–µ* –≤–æ–∑–º–æ–∂–Ω—ã–µ –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –æ—Ç–Ω–æ—à–µ–Ω–∏—è –º–µ–∂–¥—É –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏ (–≤—Å–µ –∑–æ–Ω—ã), –≤ —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ –¥–∏–∞–≥—Ä–∞–º–º—ã –≠–π–ª–µ—Ä–∞ –ø–æ–∫–∞–∑—ã–≤–∞—é—Ç —Ç–æ–ª—å–∫–æ —Ñ–∞–∫—Ç–∏—á–µ—Å–∫–∏–µ, —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–µ –æ—Ç–Ω–æ—à–µ–Ω–∏—è.

#### –ó–∞–∫–æ–Ω—ã —Ç–µ–æ—Ä–∏–∏ –º–Ω–æ–∂–µ—Å—Ç–≤

–î–ª—è –ª—é–±—ã—Ö –º–Ω–æ–∂–µ—Å—Ç–≤ `A, B, C` –∏ —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ `U` —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤—ã —Å–ª–µ–¥—É—é—â–∏–µ –∑–∞–∫–æ–Ω—ã:

*   **–ö–æ–º–º—É—Ç–∞—Ç–∏–≤–Ω–æ—Å—Ç—å (Commutative):**
    `A ‚à™ B = B ‚à™ A`
    `A ‚à© B = B ‚à© A`
*   **–ê—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω–æ—Å—Ç—å (Associative):**
    `(A ‚à™ B) ‚à™ C = A ‚à™ (B ‚à™ C)`
    `(A ‚à© B) ‚à© C = A ‚à© (B ‚à© C)`
*   **–î–∏—Å—Ç—Ä–∏–±—É—Ç–∏–≤–Ω–æ—Å—Ç—å (Distributive):**
    `A ‚à™ (B ‚à© C) = (A ‚à™ B) ‚à© (A ‚à™ C)`
    `A ‚à© (B ‚à™ C) = (A ‚à© B) ‚à™ (A ‚à© C)`
*   **–ó–∞–∫–æ–Ω—ã –¥–µ –ú–æ—Ä–≥–∞–Ω–∞ (De Morgan's Laws):**
    `¬¨(A ‚à™ B) = ¬¨A ‚à© ¬¨B` –∏–ª–∏ `(A ‚à™ B)·∂ú = A·∂ú ‚à© B·∂ú`
    `¬¨(A ‚à© B) = ¬¨A ‚à™ ¬¨B` –∏–ª–∏ `(A ‚à© B)·∂ú = A·∂ú ‚à™ B·∂ú`
*   **–ó–∞–∫–æ–Ω—ã —Ç–æ–∂–¥–µ—Å—Ç–≤–∞ (Identity Laws):**
    `A ‚à™ ‚àÖ = A`
    `A ‚à© U = A`
*   **–ó–∞–∫–æ–Ω—ã –¥–æ–ø–æ–ª–Ω–µ–Ω–∏—è (Complement Laws):**

#### 6. –î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–∞ —Å–≤–æ–π—Å—Ç–≤ –º–Ω–æ–∂–µ—Å—Ç–≤ (Proofs)
